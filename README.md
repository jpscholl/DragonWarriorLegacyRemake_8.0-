A remake of a BYOND game I originally played, developed in Dream Maker (DM).  
This project focuses on recreating core gameplay systems, mechanics, and structure while improving code organization and maintainability over previous versions.  

## Features
- Custom game mechanics implemented in Dream Maker (DM)
- Reworked legacy systems from original game version
- Event and gameplay loop structure
- Iterative redesign of core gameplay systems

## Tech Stack
- Dream Maker (BYOND)

## Purpose
This project was created as a learning exercise in game development and systems design, focusing on rebuilding and improving an existing game experience using Dream Maker.

## How to Run
1. Install BYOND
2. Open the project in Dream Maker
3. Compile and run the game

## Status
This project is under active development and represents an ongoing remake rather than a finished release.
